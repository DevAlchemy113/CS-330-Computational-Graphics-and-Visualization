This project showcases my final 3D scene created using OpenGL, including object modeling, lighting, textures, and camera controls.

## Journal Reflection

Throughout this course, my approach to designing software has improved a lot. At the beginning, I mostly focused on just getting things to work, but during this project I learned how important planning and structure are. When building my 3D scene, I had to think about how objects would be created using basic shapes and then transformed using scaling, rotation, and positioning. One of the biggest design skills I developed was breaking down complex objects, like the barrel and chest, into smaller parts and building them step by step. I followed a process of testing each piece, adjusting it, and then combining it into the final object. This approach helped me stay organized and made debugging easier. In the future, I can apply this same strategy to other projects by planning first, building in small steps, and testing as I go.

When it comes to developing programs, my approach also changed over time. I started using more iteration instead of trying to get everything perfect on the first try. For example, I had to adjust lighting multiple times to get the scene to look right, and I also made several changes to object shapes and textures. I learned that small changes can have a big impact, especially in graphics. I also got better at reading errors and fixing problems, like when the screen was completely black due to incorrect shader names. Over the milestones, I became more comfortable working with existing code instead of rewriting everything, which was important for following the assignment requirements. This experience helped me understand how real-world development works, where you build on existing systems instead of starting from scratch.

Computer science, especially computational graphics, is helping me get closer to my goals of becoming a better developer. This course gave me a strong understanding of how 3D scenes are built and rendered, which is something I had never worked on before. I learned how lighting, textures, and camera movement all work together to create a realistic environment. These skills can help me in future classes and also in professional work, especially if I work on game development, simulations, or visualization tools. Overall, this project helped me build confidence in my ability to learn complex topics and apply them in a real project, which is something I will continue to build on moving forward.
